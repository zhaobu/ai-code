# Python Game Collection

This is a collection of classic games developed using Python and Pygame, featuring multiple well-known mini games.

## Game List

- Tetris
- Snake
- Minesweeper
- Plane Battle
- Bomberman
- Breakout
- 2048
- Memory Game
- Gomoku
- Chinese Chess
- Sudoku
- Jungle Chess
- Flappy Bird
- Space Shield
- Guess Number
- Tic Tac Toe
- Tank War

## Features

- Developed with Pygame for excellent game performance and smooth operation
- Preserves core gameplay mechanics of classic games while incorporating modern interface design
- Simple and intuitive keyboard controls
- Includes score tracking and difficulty adjustment features
- Clear code structure that's easy to understand and extend

## Requirements

- Python 3.10 or higher
- Pygame 2.5.2 or higher

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/game.git
cd game
```

2. Install dependencies using Poetry:

```bash
poetry install
```

Or using pip:

```bash
pip install pygame
```

## Running the Games

```bash
python src/main.py
```

## Game Controls

Each game has its specific control scheme, but generally follows these rules:

- Arrow keys: Movement control
- Spacebar: Shoot/Confirm/Pause
- ESC: Return to main menu
- R: Restart current game

## Development Guide

The project uses Poetry for dependency management. To set up the development environment:

```bash
poetry install --with dev  # Install development dependencies
```
